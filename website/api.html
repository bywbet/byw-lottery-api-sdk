彩票API接口文档

完整API接口参考 - 实时开奖数据接口平台

接口概述

博易网彩票API接口为开发者提供实时彩票开奖数据服务，支持JSON、XML、JSON2、JSONP多种返回格式。

接口地址

http://api.byw.bet:868/api

请求方式

所有接口均使用 GET 请求方式。

请求参数

参数名 类型 必填 说明
token String 必填 API接口密钥
t String 必填 彩种代码（如：cqssc）
p String 必填 返回格式：json/xml/json2/jsonp
limit Integer 选填 返回条数（1-20，默认1）
nexttime Integer 选填 返回下一期时间（1=开启）
date String 选填 指定日期（YYYYMMDD）

响应示例

{
  "Rows": 2,
  "T": "cqssc",
  "Nexttime": "2026-04-18 22:10:00",
  "Data": [
    {
      "Expect": "20260418053",
      "Opencode": "9,0,0,1,1",
      "Opentime": "2026-04-18 21:50:03"
    }
  ]
}
