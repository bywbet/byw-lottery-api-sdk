# Byw Lottery API Documentation

## Overview

byw.bet 提供实时彩票开奖数据 API 服务。

---

## Base URL

http://api.byw.bet:868/api

---

## Example

http://api.byw.bet:868/api?token=YOUR_KEY&t=cqssc&p=json

---

## Parameters

| 参数 | 说明 |
|------|------|
| token | 接口密钥 |
| t | 彩种代码 |
| p | 返回格式 |

---

## Formats

- JSON
- XML
- JSONP
