# Web 在线演示

本目录包含彩票开奖数据API的 **Web 在线演示页面**，可直接在浏览器中运行。

## 📚 相关文档

- [SDK使用指南](../../docs/sdk-guide.md)
- [API接口文档](../../docs/api-reference.md)
- [JavaScript SDK 文档](../../src/javascript/index.js)

## 🎯 功能特性

- **在线演示** - 无需配置环境，浏览器直接打开即可体验
- **多彩种选择** - 支持多种主流彩票品种
- **实时预览** - 实时显示开奖数据模拟
- **响应式设计** - 支持桌面和移动端浏览
- **SEO优化** - 完整的 meta 标签和结构化数据

## 🚀 快速开始

### 方式一：直接打开

1. 双击 `index.html` 用浏览器打开
2. 或将整个目录部署到 Web 服务器

### 方式二：本地服务器

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .

# PHP
php -S localhost:8080
```

然后访问 `http://localhost:8080`

## 📝 使用说明

### 1. 选择彩种

从下拉菜单选择要查询的彩票品种。

### 2. 设置查询数量

输入要查询的开奖记录数量（1-20条）。

### 3. 点击查询

点击"查询开奖数据"按钮获取模拟数据。

## ⚠️ 注意事项

1. **模拟数据** - 当前演示使用浏览器端模拟数据
2. **实际对接** - 实际使用需要配置正确的 API Token
3. **跨域限制** - 如需真实调用，需要通过服务端代理
