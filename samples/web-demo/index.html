C# SDK

适用于 .NET Framework / .NET Core 项目

// NuGet安装
Install-Package BywLottery.SDK

// 或复制 src/csharp/ 到项目

示例代码：

var client = new BywClient("TOKEN");
var result = await client.GetLatestAsync("4");
// 重庆时时彩 lotteryid: 4

PHP SDK

适用于 PHP 7.0+ 项目

// Composer安装
composer require byw/lottery-api-sdk

// 或复制 src/php/ 到项目

示例代码：

$client = new BywClient('TOKEN');
$result = $client->getLatest('4');
// 重庆时时彩 lotteryid: 4

JavaScript SDK

适用于 Node.js / 浏览器环境

// npm安装
npm install byw-lottery-api-sdk

// 或复制 src/javascript/ 到项目

示例代码：

const client = new BywClient('TOKEN');
const result = await client.latest('4');
// 重庆时时彩 lotteryid: 4
